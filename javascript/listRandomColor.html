1. Backup Ekstensi
    • Buka CMD dengan administrator Wajib kalau tidak nanti access denied
    • Masuk ke folder yang untuk menyimpan hasil backup
    • Ketikkan "ASSOC >backup.txt" tanpa tanda kutip, backup adalah nama file bisa diganti sesuka kalian
    • lihat pasti sudah ada file backup.txt atau kalian memberi nama apa tadi
