Langkah

Buka WhatsappWeb lalu buka Console
  1. KLik kanan -> inspect element
  2. klik tab console
Masukkan Code diatas ke console lalu enter
Ambil List Nomor
Setelah masuk ke grup :
  1. klik kanan disitu (di nomor)
  2. inspect element
  3. Masuk tab Element dan double klik teks yang berwarna putih
    4. Klik Panah nya dulu biar kebuka, kalo ga ada nomor nya
